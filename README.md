# Student Registration

This is a demo project built with [Next.js](https://nextjs.org), designed to allow students to register through a link generated by an administrator. It also provides an admin panel to manage student data.

## Tech Stack

<p align="left">
  <img src="https://skillicons.dev/icons?i=nextjs,react,tailwind,nodejs,prisma" />
</p>

- Next.js
- React
- Tailwind CSS
- Node.js
- Prisma (ORM)

## Features

### Student

- Register using a secure link provided by the admin
- Fill in and submit personal information
- Save and update their data (if allowed)

### Admin

- Secure admin login and authentication
- Create and manage registration links for students
- Create new student records manually
- View all registered students
- Update student information
- Delete student records

### Data Management

- Export student data to **PDF**
- Export student data to **Excel (CSV/XLSX)**

## Getting Started

### 1. Clone the project

```bash
git clone <your-repo-url>
cd <your-project-name>
```

````

### 2. Install dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
```

### 3. Setup environment variables

Duplicate the example environment file:

```bash
cp .env.example .env
```

Then update the `.env` file with your database URL and other configuration.

### 4. Run database migrations (Prisma)

```bash
npx prisma migrate dev
```

(Optional) Open Prisma Studio:

```bash
npx prisma studio
```

### 5. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
````
